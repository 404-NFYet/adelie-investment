## B-1. Canvas Engine: AI 선제 제시 패턴

### 사전 연산 (Pre-compute)

**타이밍**: datapipeline 완료 직후 자동 트리거
**대상**: 오늘의 키워드별 Canvas 초기 분석 (모든 사용자 동일)

```python
# datapipeline/nodes/canvas_precompute.py (신규)
@traceable(name="canvas_precompute", tags=["canvas", "precompute"])
async def precompute_home_canvas(state: PipelineState):
    briefing = state["final_briefing"]
    analysis_md = await llm_generate(HOME_CANVAS_PROMPT, briefing)
    ctas = await generate_ctas(analysis_md, mode="home", stocks=briefing.stocks)
    chart_json = await precompute_chart(briefing.stocks[:2])
    await redis.set(f"canvas:precompute:home:{kst_today()}", json.dumps({...}), ex=86400)
```

**Redis 키**: `canvas:precompute:{mode}:{date}` (TTL 24h)

### Canvas 진입 시 첫 화면

```
1. GET /api/v1/canvas/precomputed?mode=home → Redis 캐시 확인
   HIT  → 즉시 렌더링 (0ms 지연)
   MISS → POST /api/v1/canvas/analyze (SSE 스트리밍)

2. 첫 화면 구조:
   ┌───────────────────────────┐
   │ [←] 분석 모드  진행 34%   │
   ├───────────────────────────┤
   │ ## 오늘 시장: 반도체 반등  │ ← AI 자동 생성
   │ - 핵심 포인트 1~3         │
   │ [📊 차트]                 │ ← 사전 계산 Plotly
   ├───────────────────────────┤
   │ [내 포트에 영향] [리스크]  │ ← AI 생성 CTA
   │ [과거 사례 비교]          │
   └───────────────────────────┘
```

### CTA 생성 → 실행 → 업데이트 루프

```
AI가 CTA 3개 생성 (분석 결과 기반)
  ↓
유저가 CTA 클릭 (예: "내 포트폴리오 영향")
  ↓
POST /api/v1/canvas/analyze (SSE)
  → tool_call: get_portfolio → 결과 주입
  → text_delta: 분석 스트리밍
  → visualization: 차트 JSON
  → done: { actions: [새 CTA 3개] }
  ↓
Canvas에 새 턴 추가 + 새 CTA 렌더링
(좌우 스와이프로 이전 분석 탐색 가능)
```
