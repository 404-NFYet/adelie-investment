### Step 2. 이미지를 각 LXD 서버에 배포 (docker save → lxc | docker load)

Docker Hub 푸시 없이 직접 전송.

```bash
# 이미지를 스트림으로 각 서버에 로드 (tar 임시 파일 불필요)
for srv in dev-yj99son dev-j2hoon10 dev-jjjh02 dev-ryejinn dev-hj; do

    | lxc exec "$srv" -- docker load \
    && echo "✅ $srv" || echo "❌ $srv"
done
```

### Step 4. `lxd/Makefile` `sync-lxd` 브랜치 업데이트

`sync-lxd` 타겟이 구버전(`dev/*`) 브랜치를 가리키고 있어, 향후 재사용을 위해 `dev-final/*`로 갱신.

**변경 내용:** pull 브랜치 `dev/frontend`, `dev/chatbot`, `dev/pipeline`, `dev/backend`, `dev/infra`
→ `dev-final/frontend`, `dev-final/chatbot`, `dev-final/pipeline`, `dev-final/backend`, `dev-final/infra`

커밋 메시지: `chore: sync-lxd 브랜치 dev/* → dev-final/* 갱신`
