## Issue 2: 네비바 + AgentDock 스크롤 hide/show 애니메이션

### 현황 분석

| 컴포넌트 | 위치 | z-index | 높이 | 현재 hide 방식 |
|----------|------|---------|------|---------------|
| BottomNav | `fixed bottom-0` | z-20 | `--bottom-nav-h: 68px` | 라우트/키보드 기반 `return null` |
| AgentDock | `fixed`, bottom = navH + keyboardOffset | z-30 | `--agent-dock-h: 88px` | `shouldHide` prop → `return null` |

- Framer Motion `^12.0.0` 이미 사용 중 (TutorModal, Landing 등)
- 두 컴포넌트 합산 최대 **156px**의 영역 차지
- 현재 스크롤 기반 hide/show 미구현

### 구현 계획

#### Step 1: `useScrollDirection` 훅 생성
**파일**: `frontend/src/hooks/useScrollDirection.js` (신규)

```javascript
// 스크롤 방향 감지 훅 — 임계값 이상 스크롤 시 hide 트리거
// 반환: { scrolledDown: boolean }
// - scrolledDown=true: 아래로 스크롤 중 → UI 숨기기
// - scrolledDown=false: 위로 스크롤 or 페이지 상단 → UI 표시
```

- `threshold`: 8px 이상 스크롤 시 방향 판정 (미세 진동 방지)
- `passive: true` scroll listener
- cleanup on unmount

#### Step 2: `BottomNav.jsx` 수정
**파일**: `frontend/src/components/layout/BottomNav.jsx`

1. `useScrollDirection()` 훅 추가
2. `return null` 패턴 → `motion.nav`로 전환 + `animate={{ y: scrolledDown ? '100%' : 0 }}`
3. `pointer-events: none` when hidden (접근성)

```jsx
import { motion } from 'framer-motion';

// 기존: if (hidden condition) return null;
// 변경: shouldHide면 return null (라우트 기반), 그 외엔 motion.nav로 animate

return (
  <motion.nav
    animate={{ y: scrolledDown ? '100%' : 0 }}
    transition={{ type: 'spring', damping: 30, stiffness: 300 }}
    className="fixed bottom-0 left-0 right-0 z-20 mx-auto h-[var(--bottom-nav-h,68px)] ..."
  >
    ...
  </motion.nav>
);
```

#### Step 3: `AgentDock.jsx` 수정
**파일**: `frontend/src/components/agent/AgentDock.jsx`

1. `useScrollDirection()` 훅 추가
2. 외부 wrapper `div` → `motion.div` 전환
3. 스크롤 다운 시 네비바 높이만큼 추가 translate (두 컴포넌트가 함께 사라지는 효과)

```jsx
import { motion } from 'framer-motion';

// scrolledDown 시 bottom의 navH 기여분이 없어지므로
// AgentDock도 동일량(68px) 아래로 이동해야 화면 밖으로 사라짐
const translateY = scrolledDown
  ? (hideBottomNav ? 'calc(100% + 8px)' : 'calc(100% + var(--bottom-nav-h,68px) + 8px)')
  : 0;

<motion.div
  animate={{ y: translateY }}
  transition={{ type: 'spring', damping: 30, stiffness: 300 }}
  className="pointer-events-none fixed left-0 right-0 z-30 ..."
  style={{ bottom: `calc(${hideBottomNav ? '0px' : 'var(--bottom-nav-h,68px)'} + var(--keyboard-offset,0px))` }}
>
```

**참고**: AgentDock의 `translate` 방향은 본인 높이 + (BottomNav 미표시 시 0 or navH) 만큼 아래로 이동.
두 컴포넌트 동시에 화면 밖으로 자연스럽게 퇴장.
