## Fix 2: BottomNav + TutorModal 겹침

**원인**: TutorModal이 열려도 BottomNav가 숨겨지지 않음. TutorModal(`z-50, bottom-0, 85vh`)이 BottomNav(`z-20, bottom-0, 68px`)를 덮지만, BottomNav가 여전히 렌더링되어 터치 이벤트 간섭 및 시각적 잔상 발생.

**파일**: `frontend/src/components/layout/BottomNav.jsx`

**변경**:
```jsx
// Before (line 1-2)
import { useNavigate, useLocation } from 'react-router-dom';
import useKeyboardInset from '../../hooks/useKeyboardInset';

// After
import { useNavigate, useLocation } from 'react-router-dom';
import useKeyboardInset from '../../hooks/useKeyboardInset';
import { useTutor } from '../../contexts';
```

```jsx
// Before (line 58-69) — hide 로직
export default function BottomNav() {
  const navigate = useNavigate();
  const location = useLocation();
  const { shouldHideBottomNav } = useKeyboardInset();

  if (
    HIDDEN_EXACT.includes(location.pathname) ||
    HIDDEN_PREFIXES.some((prefix) => location.pathname.startsWith(prefix)) ||
    shouldHideBottomNav
  ) {
    return null;
  }

// After
export default function BottomNav() {
  const navigate = useNavigate();
  const location = useLocation();
  const { shouldHideBottomNav } = useKeyboardInset();
  const { isOpen: isTutorOpen } = useTutor();

  if (
    HIDDEN_EXACT.includes(location.pathname) ||
    HIDDEN_PREFIXES.some((prefix) => location.pathname.startsWith(prefix)) ||
    shouldHideBottomNav ||
    isTutorOpen
  ) {
    return null;
  }
```
