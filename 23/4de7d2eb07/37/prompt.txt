## Fix 1: deploy.yml 트리거 보호 (Critical)

**파일**: `.github/workflows/deploy.yml`

### 변경 내용

**Before:**
```yaml
on:
  push:
    branches: [develop]

env:
  TAG: latest
```

**After:**
```yaml
on:
  push:
    branches: [main]          # develop 자동배포 제거 → main 머지 시만 자동배포
  workflow_dispatch:           # 수동 트리거 추가
    inputs:
      tag:
        description: '배포할 Docker 이미지 태그 (예: latest, feb20-stable)'
        required: false
        default: 'latest'
      skip_build:
        description: '빌드 스킵 (기존 이미지 배포 시 true)'
        required: false
        default: 'false'

env:
  TAG: ${{ github.event.inputs.tag || 'latest' }}
```

### 추가로 수정할 부분 (workflow body)

빌드 단계에 조건 추가 — `skip_build: 'true'`이면 빌드/push를 건너뜀:
```yaml
- name: Build and push frontend
  if: github.event.inputs.skip_build != 'true'
  # ... 기존 빌드 step
```

배포 단계: TAG 환경변수를 docker compose에 전달:
```yaml
- name: Deploy to production
  run: |
    ssh ... '
      cd ~/adelie-investment
      git fetch origin
      git checkout release/feb20-stable  # 또는 ${{ env.DEPLOY_BRANCH }}
      TAG=${{ env.TAG }} docker compose -f docker-compose.prod.yml pull frontend backend-api
      TAG=${{ env.TAG }} docker compose -f docker-compose.prod.yml up -d --no-deps frontend backend-api
    '
```

> **실용적 접근**: 배포 브랜치를 `workflow_dispatch.inputs.branch`로도 받을 수 있지만,
> 일단 최소 변경(develop 트리거 제거 + workflow_dispatch 추가 + TAG 파라미터화)만 적용.

## Fix 2: lxd/Makefile rollback-stable 타겟 추가 (Optional)

**파일**: `lxd/Makefile`

적절한 위치(기존 `deploy-test` 타겟 근처)에 추가:

```makefile
## 금요일 안정 버전(feb20-stable)으로 빠른 롤백
rollback-stable:
	@echo "$(CYAN)▶ 롤백: feb20-stable → deploy-test$(RESET)"
	ssh $(DEPLOY_HOST) ' \
	  cd ~/adelie-investment && \
	  git fetch origin && \
	  git checkout release/feb20-stable && \
	  TAG=feb20-stable docker compose -f docker-compose.prod.yml pull frontend backend-api && \
	  TAG=feb20-stable docker compose -f docker-compose.prod.yml up -d --no-deps frontend backend-api && \
	  docker exec adelie-frontend nginx -s reload 2>/dev/null || true && \
	  echo "✅ 롤백 완료: $$(docker ps --format \"{{.Image}}\" | grep adelie)" \
	'
	@echo "$(GREEN)✅ rollback-stable 완료$(RESET)"
```

`DEPLOY_HOST` 변수가 lxd/Makefile에 이미 정의되어 있으므로 그대로 활용.
