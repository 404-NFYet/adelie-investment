# Phase 6: 복습 카드 시스템

## 문제

사용자가 AI 튜터와 나눈 대화를 나중에 복습하기 어려웠습니다. 단순 저장으로는 핵심을 파악하기 힘듭니다.

## 해결

### 1. 복습 카드 구조화
대화 내용을 모듈화하여 저장:
- **핵심 개념**: 배운 주요 용어/개념
- **당시 상황**: 대화 맥락 (날짜, 주제)
- **핵심 포인트**: 중요 인사이트
- **요약 스니펫**: 전체 대화 요약

### 2. 카드 뷰어 UI (`ReviewCardViewer.jsx`)
- 스와이프로 카드 넘기기
- 그라데이션 헤더 (주황색)
- 섹션별 아이콘과 배경색 구분
- "이어서 대화하기" CTA

### 3. 저장 UX 개선
- "저장" → "복습 카드로 저장하기" 문구 변경
- 직관적인 CTA

### 4. HTML 포맷팅
카드를 예쁜 HTML로 저장하여 언제든 열람 가능
