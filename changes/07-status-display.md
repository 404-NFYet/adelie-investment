# Phase 7: 상태 표시 개선

## 문제

에이전트가 응답 중인지, 완료했는지 사용자가 명확히 알 수 없었습니다.

## 해결

### 1. 상태 인디케이터 (`AgentChatInput.jsx`)

| 상태 | 아이콘 | 색상 | 텍스트 |
|------|--------|------|--------|
| idle | ✏️ | 회색 | 무엇이든 물어보세요 |
| thinking | ⏳ | 파랑 | 생각 중... |
| tool_call | 🔧 | 보라 | 도구 실행 중... |
| streaming | ✍️ | 주황 | 답변 작성 중... |
| answering | ✍️ | 주황 | 답변 작성 중... |
| done | ✅ | 초록 | 답변 완료 |
| error | ❌ | 빨강 | 오류가 발생했어요 |

### 2. 입력 필드 상태 연동
- 응답 중: 입력 비활성화, 스피너 표시
- 완료 후: 입력 활성화, 전송 버튼 복원

### 3. 시각적 피드백
- 스트리밍 중 커서 깜빡임 애니메이션
- 상태 텍스트 위 점 표시로 상태 구분
