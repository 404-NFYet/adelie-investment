## 디자인 원칙 (공통)

시작하기 전에, 세 컨셉 모두 아래 원칙을 공유합니다.

| 원칙 | 설명 |
|------|------|
| **미션 퍼스트** | 홈 진입 시 가장 먼저 "왜 아델리를 써야 하는가"가 눈에 들어와야 함 |
| **루프 가시화** | 교육 → 퀴즈 → 모의투자 순환 구조가 시각적으로 보여야 함 |

## 컨셉 A: "학습 루프 가이드" 🔄

> **핵심 아이디어**: 홈화면 자체가 **3단계 학습 루프의 네비게이션 허브**가 되는 것.  
> 유저가 "다음에 뭘 해야 하지?"를 항상 알 수 있게.

### 섹션별 상세

#### ① 히어로 카드: 미션 메시지
- Landing에서 전달한 "쉽고 깊은 금융 이야기"의 **연장선**
- 매일 다른 동기부여 메시지 로테이션 가능 (시간대별/학습 단계별)
- 오렌지 그라데이션 유지 → 브랜드 연속성
- **현재 보유자산 카드 제거**, 대신 미션 메시지로 교체

#### ② 학습 루프 3스텝 (핵심 차별점)
- 가로 스크롤형 3장 카드 (브리핑 → 퀴즈 → 모의투자)
- 각 카드에 **현재 상태 표시**:
  - 완료: ✅ 체크 + 연한 초록 테두리
  - 진행 가능: 🟠 활성 + "시작하기" CTA
  - 잠금: 🔒 회색 + "브리핑을 먼저 읽어보세요"
- 화살표 연결로 **순서가 있다는 것**을 시각적으로 전달
- 각 카드 탭 시: 교육탭/퀴즈모달/포트폴리오로 바로 이동

#### ③ 자산 요약 바
- 현재의 거대한 오렌지 카드 대신, **한 줄짜리 슬림 바**
- "💰 내 투자 현황 12,450원 ›" — 탭하면 포트폴리오로 이동
- 보유 자산 정보는 유지하되 시각적 비중을 10→2로 축소

#### ④ 동기부여 푸터
- 연속 학습일, 이번 주 활동량 등 **한 줄 격려**
- 현재의 주간 캘린더 섹션을 이 한 줄로 압축

### 장단점

| 장점 | 단점 |
|------|------|
| ✅ "다음에 뭘 해야 하지?"가 명확 | ❌ 3단계가 항상 순서대로여야 하는지 유저 부담 |
| ✅ 교육-모의투자 루프가 시각적으로 보임 | ❌ 브리핑이 없는 날은 Step 1이 비어보임 |
| ✅ 중복 제거가 자연스러움 | ❌ 히어로 메시지가 반복되면 무시될 수 있음 |

### 섹션별 상세

#### ① 에디토리얼 히어로: 오늘의 이야기
- 현재 카드뉴스 3장 → **가장 매력적인 1장만 크게**
- 매거진/뉴스레터 느낌: 큰 이미지 + 큰 타이틀 + 요약 문구
- "이야기 읽으러 가기" CTA → `Narrative` 페이지로 직행
- 나머지 카드뉴스는 교육탭에서만 볼 수 있음 → **교육탭 방문 유도**
- **홈에서 카드뉴스 리스트 중복 제거**, 대표 1건만 노출

#### ② 학습 여정 트래커
- 주간 캘린더를 **프로그레스 바 + 요일 뱃지**로 변환
- 같은 데이터(activitiesByDate)지만 **훨씬 컴팩트**한 표현
- 교육탭의 월간 캘린더와 **명확히 다른 형태** → 중복 해소

#### ③ 2-컬럼 액션 카드
- 퀴즈: DailyQuizMissionCard를 **축소 버전**으로 (이름 + 상태만)
- 모의투자: 보유 자산 요약 + 진입 CTA
- 탭하면 각각 퀴즈 모달 / 포트폴리오로 이동

#### ④ 미션 한마디
- Landing에서 쓰인 핵심 카피를 재활용
- 화면 가장 아래, 소극적이지만 **철학이 스며드는** 효과

### 장단점

| 장점 | 단점 |
|------|------|
| ✅ "읽을거리가 있네!"라는 즉각적 인상 | ❌ 오늘의 카드뉴스가 1건도 없으면 히어로가 비어보임 |
| ✅ 매거진 느낌이 교육 앱의 정체성과 맞음 | ❌ 콘텐츠 품질에 크게 의존 |
| ✅ 교육탭 방문 유도가 자연스러움 | ❌ 모의투자 진입이 상대적으로 약해짐 |

> 매일 완료해야 할 미션 체크리스트처럼 구성. 학습을 "클리어"하는 경험.

### 섹션별 상세

#### ① 미션 배너: 브랜드 스테이트먼트
- 딥 네이비/다크 배경 + 오렌지 포인트
- Landing의 "쉽고 깊은 금융 이야기" 그대로 재사용
- **앱을 열 때마다 미션을 상기**시킴
- 스파클 애니메이션으로 프리미엄 느낌

#### ② 미션 보드: 세로 체크리스트 (핵심 차별점)
- 세로 점선 타임라인 + 3단계 카드
- **상태 기반 UI**:
  - ✅ 완료 — 초록 체크, 연한 배경, 축소
  - 🟠 현재 — 오렌지 보더, 확대, `[GO →]` 버튼 활성
  - 🔒 다음 — 회색, "이전 단계를 완료하세요" 또는 자유 진입 허용
- 잠금을 **하드 잠금이 아닌 소프트 잠금**으로 → 탭은 되지만 안내 메시지 표시
- 퀴즈/브리핑 완료 여부는 `useDailyQuiz` + `useActivityFeed`에서 가져옴

#### ③ 슬림 자산 바 (컨셉 A와 동일)

#### ④ 스트릭 & 주간 프로그레스
- 연속 학습일, 주간 달성률을 **한 줄로 압축**
- 포인트 시스템/뱃지 추가 가능성 (확장 포인트)

### 장단점

| 장점 | 단점 |
|------|------|
| ✅ 게이미피케이션 → 완료 욕구 자극 | ❌ 너무 "숙제" 느낌이 들 수 있음 |
| ✅ "다음 뭐 하지?"가 가장 명확 | ❌ 잠금 UX가 답답할 수 있음 |
| ✅ 미션 배너가 브랜드를 강화 | ❌ 3단계가 고정이면 유연성 감소 |
| ✅ 진행률이 보여 중독성 있음 | ❌ 구현 복잡도가 가장 높음 |

## 🔥 심화 기획: "오늘의 브리핑" 요소 구성 (D안 기반)

사용자가 이 섹션을 단순한 "뉴스 목록"이 아니라 **"나를 위해 큐레이션 된 필수 미션"**으로 느끼게 하려면 어떤 요소가 들어가야 할까?

### 1. 배지와 메타데이터 (유도 장치)
단순 제목만 보여주는 것을 넘어 클릭을 유도하는 장치 필요.
- **카테고리 뱃지**: `시황`, `종목분석`, `전략`, `마인드셋` 등 명확한 토픽 컬러링 (토스식)
- **소요 시간 (Time to Read)**: "3분 읽기", "5분 읽기" 표시로 진입 장벽 낮춤
- **서브타이틀**: 기사 본문을 클릭하지 않아도 어떤 내용인지 알 수 있는 한 줄 핵심 요약 (예: "PER 15배, 분할 매수 전략")

### 2. 읽음/완료 상태 시각화 (Gamification)
단순한 기사 목록이 아니라 '미션'이 되어야 함.
- 3개의 브리핑을 모두 읽으면 메인 펭귄의 표정이 변하거나 불꽃(스트릭) 뱃지에 이펙트 발생
- 읽은 기사는 브리핑 리스트에서 회색조(Dimmed) 처리되거나 체크크마크(✅) 표시
- "오늘의 브리핑 (3가지)" 뱃지가 (1/3 읽음) 형태로 실시간 진행률 반영

### 3. 브리핑 후속 액션 연결 (The Loop)
브리핑 리스트의 맨 마지막 카드는 자연스럽게 다음 단계(퀴즈/투자)로 넘어가게 유도.
- 브리핑 3개를 다 읽으면 맨 아래 숨겨져 있던 **[🏆 오늘의 퀴즈 풀기]** 카드가 활성화되어 팝업처럼 등장
- 해당 브리핑(예: 삼성전자)을 읽고 나면 하단에 작게 **[`삼성전자` 모의투자 해보기 〉]** 버튼 노출

## 🤔 추가 고민: "배운 걸 써먹었는지" 어떻게 알 수 있을까? (The Loop Completion)

사용자가 단순히 모의투자 탭에 들어갔다고 해서 "학습한 내용을 적용했다"고 보기는 어렵습니다. 이를 데이터로 증명하고, 유저에게 피드백을 주려면 앱 내부적으로 **연결 고리(Linkage)**가 필요합니다.

### 기획 제안: [태그/종목 단위의 매칭 시스템]

1. **데이터 맵핑 (Data Mapping)**
   - `오늘의 브리핑` 카드는 특정 키워드(예: `#삼성전자`, `#반도체`, `#금리`) 메타데이터를 가집니다.
   - 유저가 모의투자(Portfolio.jsx)에서 매수/매도 필터를 실행할 때, 해당 거래의 `종목명`이나 `섹터`가 브리핑의 키워드와 일치하는지 백그라운드에서 체크합니다.

2. **UI 피드백 (Reward)**
   - 일치하는 거래가 발생하면, 홈 화면이나 모의투자 탭 상단에 특별한 토스트 메시지를 띄웁니다.
   - *"🎉 방금 배운 `반도체` 지식을 실제 투자에 적용하셨네요! (학습 포인트 +50p)"*
   - 홈 화면의 '연속 학습' 스트릭 배지가 단순 출석 불꽃(🔥)에서 **'실전 적용 불꽃(💥)'** 등 특별한 아이콘으로 업그레이드되게 만듭니다.

3. **기대 효과**
   - **유저 입장**: 내 학습이 헛되지 않고 '투자 감각'으로 이어지고 있다는 강력한 효능감(Self-efficacy)을 얻습니다.
   - **PM 입장**: '교육 탭 진입률'이라는 1차원적 지표를 넘어, **'교육 연계 투자 발생률(Education-Linked Trade Rate)'** 이라는 훨씬 강력하고 매력적인 핵심 지표(North Star Metric)를 측정할 수 있게 됩니다.
