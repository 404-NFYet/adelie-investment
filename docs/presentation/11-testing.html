def test_rrf_calculation(self):

    assert abs(score_rank1 - 0.01639) < 0.001

TestClient 패턴

모든 백엔드 테스트는 TestClient(app, raise_server_exceptions=False) 패턴을 사용합니다. raise_server_exceptions=False로 설정하면, 서버 내부 에러가 예외를 던지는 대신 HTTP 500 응답으로 반환되어 실제 운영 환경의 동작을 더 정확히 재현합니다.

@pytest.fixture(scope="module")
def client():
    from app.main import app
    return TestClient(app, raise_server_exceptions=False)

Backend 테스트 서비스

test-backend 서비스는 FastAPI Dockerfile을 기반으로 빌드되며, 테스트 의존성을 설치한 후 pytest를 실행합니다. depends_on의 condition: service_healthy로 DB가 준비될 때까지 대기합니다.

test-backend:
  build:
    context: .
    dockerfile: fastapi/Dockerfile
  command: >
    sh -c "pip install pytest pytest-asyncio httpx -q &&
           pytest tests/ -v --tb=short --junitxml=/app/test-results/backend.xml"
  environment:
    DATABASE_URL: postgresql+asyncpg://test:test@postgres-test:5432/test_narrative
    REDIS_URL: redis://redis-test:6379/0
    OPENAI_API_KEY: ${OPENAI_API_KEY:-test}
    ENVIRONMENT: test
  volumes:
    - ./tests:/app/tests
    - ./test-results:/app/test-results
  depends_on:
    postgres-test:
      condition: service_healthy
    redis-test:
      condition: service_healthy

  environment:
    BASE_URL: http://frontend:80

Q9 비동기 테스트 패턴(pytest-asyncio)의 설정과 사용 방법을 설명해 주세요. +

Adelie Investment의 백엔드는 비동기 FastAPI 기반이므로, 테스트에서도 비동기 패턴을 지원해야 합니다. pytest-asyncio의 auto 모드를 사용하여 이를 간편하게 처리합니다.

asyncio_mode = auto 설정

pytest.ini에서 asyncio_mode = auto로 설정하면, async def로 정의된 테스트 함수를 자동으로 비동기 테스트로 인식합니다.

[pytest]
asyncio_mode = auto

이 설정 덕분에 @pytest.mark.asyncio 데코레이터를 명시적으로 붙이지 않아도 됩니다. 다만, 명시적으로 사용해도 문제없습니다.

비동기 테스트 작성 예시

tests/unit/test_stock_price_service.py에서 async def를 사용한 비동기 테스트:

@pytest.mark.asyncio
async def test_get_current_price_uses_cached_value_first(monkeypatch):
    cached = json.dumps({
        "stock_code": "005930",
        "stock_name": "삼성전자",
        "current_price": 71000,
        "change_rate": 1.23,
        "source": "kis",
    })

    fake_cache = _FakeCache(_FakeRedisClient(cached=cached))

    async def _fake_get_redis_cache():
        return fake_cache

    monkeypatch.setattr(stock_price_service, "get_redis_cache", _fake_get_redis_cache)

    result = await stock_price_service.get_current_price("005930")
    assert result["current_price"] == 71000

동기 TestClient vs 비동기 AsyncClient

방식사용 상황예시
TestClient (동기)대부분의 API 통합 테스트client.get("/api/v1/health")
httpx.AsyncClient (비동기)SSE 스트리밍, 동시 요청 테스트async with AsyncClient(app=app) as ac:

FastAPI의 TestClient는 내부적으로 비동기 엔드포인트를 동기적으로 호출할 수 있어, 대부분의 테스트에서는 TestClient만으로 충분합니다.

conftest.py에서의 마커 등록

def pytest_configure(config):
    """pytest-asyncio 자동 모드 설정."""
    config.addinivalue_line("markers", "asyncio: mark test as async")

asyncio 마커를 명시적으로 등록하여 pytest의 "unknown marker" 경고를 방지합니다.

주의 사항

  • auto 모드에서는 모든 async def test_* 함수가 자동으로 비동기 실행됩니다.
  • event loop은 pytest-asyncio가 관리하므로, 별도의 asyncio.run() 호출이 불필요합니다.
  • Fixture도 async def로 정의할 수 있으며, auto 모드에서 자동 감지됩니다.
